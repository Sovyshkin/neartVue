# Дополнительные поля для картин

В модель "Картина" (Artwork) добавлены следующие новые поля для редактирования дополнительной информации:

## Новые поля:

### 1. aboutArtwork (О произведении)
- **Тип:** Текст (до 1000 символов)
- **Описание:** Информация о произведении, его создании, особенностях
- **Дефолтный текст:** "Это произведение создано с особым вниманием к деталям и цветовой гармонии. Художник использовал уникальную технику, которая позволяет передать глубину и эмоциональную насыщенность сюжета."

### 2. technique (Техника исполнения)
- **Тип:** Текст (до 1000 символов)
- **Описание:** Описание техники и материалов, использованных при создании произведения
- **Дефолтный текст:** "Работа выполнена в традиционной технике с использованием высококачественных материалов, что обеспечивает долговечность и сохранение первоначального вида произведения на долгие годы."

### 3. storageConditions (Условия хранения)
- **Тип:** Текст (до 1000 символов)
- **Описание:** Рекомендации по хранению и уходу за произведением
- **Дефолтный текст:** "Рекомендуется размещать вдали от прямых солнечных лучей и источников влажности. Оптимальная температура хранения 18-22°C при влажности 40-60%."

### 4. deliveryInfo (Доставка и упаковка)
- **Тип:** Текст (до 1000 символов)
- **Описание:** Информация о доставке и упаковке произведения
- **Дефолтный текст:** "Произведение будет тщательно упаковано в специальную защитную упаковку и доставлено курьерской службой с возможностью отслеживания."

## Как это работает:

1. **В админ-панели Strapi** вы можете заполнить эти поля для каждой картины
2. **Если поле заполнено** - на сайте отображается ваш текст
3. **Если поле пустое** - отображается дефолтный текст

## Инструкция по использованию:

1. Откройте админ-панель Strapi: http://localhost:1337/admin
2. Перейдите в раздел "Картины" (Artworks)
3. Выберите картину для редактирования или создайте новую
4. Найдите новые поля в форме редактирования:
   - О произведении (aboutArtwork)
   - Техника исполнения (technique)
   - Условия хранения (storageConditions)
   - Доставка и упаковка (deliveryInfo)
5. Заполните нужные поля или оставьте пустыми для использования дефолтных текстов
6. Сохраните изменения

Изменения сразу отобразятся на странице картины на сайте.
